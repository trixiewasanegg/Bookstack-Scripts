# Bookstack-Scripts
Misc bookstack scripts


# Dependencies
- Requests
- [Python dotenv](https://pypi.org/project/python-dotenv/)

# Dotenv config
```
TOKEN_ID=[Token ID from your API key]
TOKEN_SECRET=[Token Secret from your API key]
ENDPOINT=[URL for the API endpoint (usually https://domain/api)]
SITEMAP_ID=[Page ID for the SiteMap]
```
